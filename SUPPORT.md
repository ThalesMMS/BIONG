# Support

This is an experimental research codebase with limited support capacity.

## Usage And Interpretation Questions

Open a GitHub issue with a `[Question]` prefix. Include the command, output or
artifact, and the [README.md](README.md) or [docs/](docs/) page you checked.

## Bug Reports

Use the bug report issue template. Include reproduction steps, expected
behavior, actual behavior, Python version, operating system, and relevant
output, summary fields, or trace snippets.

## Feature Requests And Research Ideas

Use the feature request / research idea issue template. Explain:

- the motivation
- the expected research or reproducibility value
- alternatives considered
- whether the change affects CLI output, benchmarks, interfaces, or saved
  artifacts

## Response Expectations

Responses are best-effort. This project does not provide guaranteed support
timelines.
